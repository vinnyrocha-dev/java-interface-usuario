# Java - Interface de Usuário

Projeto simples demonstrando o uso de **interfaces em Java**.

## Funcionalidades

O programa possui um menu no terminal que permite:

1 - Cadastrar usuário  
2 - Mostrar usuário  
3 - Atualizar nome  

## Conceitos utilizados

- Interface
- Implementação de interface
- Encapsulamento
- Scanner
- Estrutura de decisão (switch)

## Estrutura do projeto

- IUsuarioService → interface com os métodos
- Usuario → implementação da interface
- Main → menu principal do programa
